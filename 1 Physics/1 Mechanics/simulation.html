Projectile Motion Simulation

This document presents a Python script that simulates projectile motion, calculating the range as a function of the angle of projection and visualizing a sample trajectory. The simulation assumes no air resistance and constant gravitational acceleration.

Python Code

The following code uses NumPy for calculations and Matplotlib for plotting. It computes the range for different initial velocities and plots a trajectory for a specific case.


import numpy as np
import matplotlib.pyplot as plt

# Constants
g = 9.81  # m/s^2
v0_values = [10, 20, 30]  # Initial velocities (m/s)
theta_deg = np.arange(0, 91, 1)  # Angles from 0 to 90 degrees
theta_rad = np.deg2rad(theta_deg)

# Range function
def range_projectile(v0, theta_rad, g=9.81):
    return (v0**2 * np.sin(2 * theta_rad)) / g

# Plot range vs angle for different v0
plt.figure(figsize=(10, 6))
for v0 in v0_values:
    R = range_projectile(v0, theta_rad)
    plt.plot(theta_deg, R, label=f'v0 = {v0} m/s')

plt.xlabel('Angle of Projection (degrees)')
plt.ylabel('Range (m)')
plt.title('Range vs. Angle of Projection')
plt.legend()
plt.grid(True)
plt.show()

# Simulate a single trajectory (example: v0 = 20 m/s, theta = 45 deg)
v0 = 20
theta = np.deg2rad(45)
t_flight = (2 * v0 * np.sin(theta)) / g
t = np.linspace(0, t_flight, 100)
x = v0 * np.cos(theta) * t
y = v0 * np.sin(theta) * t - 0.5 * g * t**2

plt.figure(figsize=(10, 6))
plt.plot(x, y)
plt.xlabel('x (m)')
plt.ylabel('y (m)')
plt.title('Projectile Trajectory (v0 = 20 m/s, θ = 45°)')
plt.grid(True)
plt.show()

Explanation

Range Calculation

The function range_projectile computes the horizontal range using the formula:

\[ R = \frac{v_0^2 \sin(2\theta)}{g} \]

where:

  • \( v_0 \): Initial velocity (m/s)
  • \( \theta \): Angle of projection (radians)
  • \( g \): Gravitational acceleration (9.81 m/s²)

The script plots the range for initial velocities of 10, 20, and 30 m/s across angles from 0° to 90°.

Trajectory Simulation

For a specific case (\( v_0 = 20 \, \text{m/s} \), \( \theta = 45^\circ \)), the script calculates the time of flight and generates the parabolic trajectory using the equations:

\[ x(t) = v_0 \cos(\theta) t \]

\[ y(t) = v_0 \sin(\theta) t - \frac{1}{2} g t^2 \]

Expected Output

The script produces two plots:

  1. Range vs. Angle: A graph showing range vs. angle of projection for different initial velocities. Each curve peaks at 45°, confirming the theoretical maximum range.
  2. Trajectory: A parabolic path for \( v_0 = 20 \, \text{m/s} \), \( \theta = 45^\circ \), showing the projectile’s motion from launch to landing.
